One word. Zero words spoken. Just your face. Teams shout it out.

Free · No ads · No accounts · A room is required to play

How to Play

  1. Host a Game — pick a language (Tagalog or English), your categories, team names, whether Letter Hints are on, and an optional Time per Round. Tap Start Room to get a 4-letter room code.
  2. Join as Display — on the TV or laptop everyone can see, open this same page and enter the Host's room code.
  3. Start Game — the Host taps Start once the Display is connected.
  4. Each round, the Host privately shows the word or phrase to whoever is acting for the round — pass the phone to them, or turn the screen away from the rest of the group. Nobody else should see it.
  5. Face only — the actor conveys the word using facial expression alone. No hands, no body, no props, no sounds. Their team shouts out guesses as soon as they think they know it.
  6. The Display never shows the word or any picture clue — just blank letter tiles, the category, the timer, and both scores.
  7. The Host taps Team A got it / Team B got it to award the point and deal the next round, or Skip if nobody can get it.
  8. If Letter Hints are on, the Host can Reveal a Letter at any time to help a stuck team.
  9. If a timer is set, the Host taps Start Timer when the next actor is ready. Time running out auto-skips the round (no point awarded) — the next one waits, paused, until Start Timer is tapped again.
  10. The game ends when the deck runs out — highest score wins (a tie is a draw).

A working room is required to play: the Host screen shows the answer, so it can't double as the shared Display.
